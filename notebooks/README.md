# ПРОЕКТ: ПРОГНОЗИРОВАНИЕ ОТТОКА КЛИЕНТОВ ТЕЛЕКОМ-КОМПАНИИ

## ЦЕЛЬ ПРОЕКТА
Разработать модель машинного обучения для предсказания оттока клиентов телеком-компании.

## ОСНОВНЫЕ РЕЗУЛЬТАТЫ
- Точность лучшей модели: 80.2%
- Самый важный фактор оттока: тип контракта
- Клиенты с помесячным контрактом уходят в 14 раз чаще
- Главная рекомендация: переводить клиентов на долгосрочные контракты

## КАК ЗАПУСТИТЬ ПРОЕКТ
1. Установите библиотеки: pip install pandas numpy matplotlib seaborn scikit-learn jupyter
2. Запустите Jupyter: jupyter notebook
3. Откройте файл: analysis.ipynb
4. Запустите все ячейки: Cell -> Run All
